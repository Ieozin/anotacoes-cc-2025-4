# Tema Extra: Introdução a Redes
**Status:** Concluído

## 1. Conceitos
* **Rede:** Dispositivos conectados compartilhando recursos.
* **Evolução:** Mainframes -> ARPANET (Militar) -> Internet Global.

## 2. Comutação (O Conceito Mais Importante)
Como os dados viajam.
* **Circuitos (Telefonia antiga):** Cria um caminho físico dedicado.
    * Prós: Estável. Contras: Se não falar, desperdiça linha.
* **Pacotes (Internet):** Quebra a info em pedaços. Não reserva caminho.
    * Prós: Otimiza a rede, rotas dinâmicas. Contras: Pode chegar fora de ordem (TCP resolve isso).

## 3. Classificação
* **Topologia:** Desenho da rede (Estrela, Anel, Barramento).
    * **Malha (Mesh):** A mais robusta (internet), vários caminhos possíveis.
* **Escala:** PAN (Pessoal) -> LAN (Local) -> MAN (Cidade) -> WAN (Mundo).

## 4. Tendências
* **IoT:** Internet das Coisas. Tudo conectado.
* **SDN (Software Defined Networks):** Separar o cérebro (controle) do músculo (hardware). Gerência centralizada.
